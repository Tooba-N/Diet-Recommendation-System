# Diet Recommendation System

This project provides a Diet Recommendation System that helps users find the best diet plan based on their age, gender, weight, height, and activity level. It includes a Streamlit application for user interaction and a Jupyter notebook for data analysis and model building.

## Project Structure

- **app2.py**: Streamlit application for user interaction.
- **main (1).py**: Jupyter notebook for data analysis and model building.

## Features
- BMI Calculation: Calculate Body Mass Index (BMI) based on weight and height.
- BMR Calculation: Calculate Basal Metabolic Rate (BMR) based on gender, weight, height, and age.
- Daily Calorie Needs: Calculate daily calorie needs based on activity level.
- Meal Recommendations: Generate meal recommendations based on nutritional content using a Random Forest Regressor model.
